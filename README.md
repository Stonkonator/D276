CODY NASON PORTFOLIO WEBSITE
=============================

This is a portfolio project submission for the WGU class D276, and includes the following:

HTML FILES (3 required pages):
1. resume.html - Complete HTML-based résumé
2. cover-letter.html - Professional cover letter
3. career-goals.html - Career goals and vision
4. index.html - Home page with projects section

CSS FILE:
- style.css - External stylesheet with modern, sleek design

MEDIA FILES:
- headshot.png - Professional headshot image
- background-music.mp3 - Original music composition
- project-demo.mp4 - Project walkthrough video

All pages include:
- Proper HTML structure (html, head, title, body)
- Header, nav, main, section, footer elements
- At least three heading elements (h1-h6)
- Paragraph, span, line break elements
- Horizontal rules for visual separation
- Div elements with the current date using JavaScript
- Hyperlinks between pages
- Ordered and unordered lists
- Tables (on resume page)
- CSS styling linked to all HTML pages
